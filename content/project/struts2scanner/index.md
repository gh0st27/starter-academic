Apache Struts is a free, open-source framework for creating elegant, modern Java web applications. It has its share of critical vulnerabilities, with one of its features, OGNL â€“ Object-Graph Navigation Language, being at the core of many of them.

Struts2Scanner is a vulnerability scanner to find out if a target endpoint is vulnerable to Remote Code Execution. Currently it checks against following vulnerabilities.

* cve-2020-17530
* cve-2019-0230
* cve-2018-11776
* cve-2017-5638
* cve-2017-9791
